# 📞 Lista de Contactos - Proyecto de Ejemplo

## 📖 Descripción

Este proyecto es una página web que simula una lista de contactos. Permite agregar nuevos contactos mediante un formulario donde se ingresan el nombre, apellido y número de teléfono. Además, muestra una lista con todos los contactos agregados. como parte de mi portafolio de certificación Full Stack.

## 📈 Características
- **Formulario de Contacto**: Incluye campos para nombre, apellido y número de teléfono.
- **Botón de Agregar Contacto**: Simula un boton que permite agregar nuevos contactos a la lista.
- **Botón de Editar Contacto**: Simula un boton que permite editar datos de los contactos a la lista.
- **Botón de Eliminar Contacto**: Simula un boton que permite eliminar contactos a la lista.
- **Lista de Contactos**: Muestra todos los contactos agregados.

## 📚 Tecnologías Utilizadas
- **HTML**: Estructura de la página.
- **CSS**: Estilos visuales.

## 📂 Estructura del Proyecto
El proyecto consta de los siguientes archivos:

- `index.html`: Este archivo contiene la estructura básica de el formulario para agregar un contacto y la lista de contactos.
- `index.css`: Este archivo contiene todos los estilos CSS aplicados a la pagina.
- `global.css`: Este archivo contiene etiquetas globales como colores para tema claro/oscuro y estilo general.
